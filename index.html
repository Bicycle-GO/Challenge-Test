탄감자

자전거 챌린지와 탄소포인트를 한 흐름으로

앱 사용자는 주행하고 명소를 인증하며 포인트를 쌓고, 운영자는 데이터 검증과 정산 승인을 처리합니다.

1 주행 기록

거리, 시간, 평균 속도, 경로를 저장합니다.

2 CO2 산정

자동차 대체 이동 기준으로 절감량을 계산합니다.

3 포인트 적립

절감량과 챌린지 보너스를 탄소포인트로 환산합니다.

4 관리자 승인

QR, GPS, 이상 주행 여부를 검토한 뒤 정산합니다.

기본 산정식

예상 CO2 절감량 = 주행거리(km) × 0.192kgCO2e

탄소포인트 = 절감량(kgCO2e) × 100P + 명소/배지 보너스

웹 서버 모드

Node 서버가 정적 앱과 `/api`를 함께 제공해 주행 기록, QR 체크인, 검증 요청을 저장합니다.

PWA/GPS 모드

브라우저 위치 권한을 받아 거리와 시간을 계산하고 서버로 샘플을 전송합니다.

Android APK 모드

WebView, 내부 저장소, Foreground Service 구조로 화면 꺼짐 GPS 기록까지 확장했습니다.

웹 서버로 열기 API 상태 보기

웹에서는 `http://localhost:4173` 주소로 열 때 서버 저장과 PWA 기능이 가장 안정적으로 동작합니다.
